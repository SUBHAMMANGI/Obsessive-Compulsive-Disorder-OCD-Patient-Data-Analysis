# Obsessive-Compulsive Disorder (OCD) Patient Data Analysis

## Overview
This project focuses on analyzing a dataset of 1,500 OCD patients to provide actionable insights for OCD treatment decisions. The analysis was done using SQL for data manipulation and feature engineering, enhancing data quality for better insights. Interactive dashboards were created using Power BI to visualize clinical and demographic data, offering a clear understanding of patterns and trends in the dataset.

## Key Features
- **Data Source**: Analysis conducted on a dataset of 1,500 OCD patients.
- **Data Processing**: Utilized SQL for data manipulation and feature engineering to enhance data quality and extract valuable insights.
- **Visualization**: Developed interactive Power BI dashboards to visualize trends, gender and ethnic distribution, compulsive behaviors, and obsession scores.

## Dashboard Highlights
- **Patient Trends**: Monthly analysis of patient count trends over time.
- **Demographic Distribution**: Gender and ethnicity segmentation for better understanding of patient demographics.
- **Compulsion Types**: Visualization of common compulsive behaviors such as washing, counting, and checking.
- **Obsessions vs. Scores**: Correlation between obsession types and their average scores.

## Technologies Used
- **SQL**: For data manipulation and feature engineering.
- **Power BI**: For building interactive dashboards and data visualization.
